Was ist die OFDB ?

Ein Auszug aus Wikipedia beschreibt es wohl am besten:

Die Online-Filmdatenbank (OFDb) ist eine deutschsprachige Datenbank über nationale und internationale Filme. Sie bietet kostenlos grundlegende Informationen über Kinofilme, Fernsehfilme und Fernsehserien. Dazu gehören deutscher Titel, Original- bzw. Alternativ-Titel, Erscheinungsjahr, Regisseur, Schauspieler, Genre, Fortsetzungen, Fassungen und eine kurze Inhaltsbeschreibung.

Zum Artikel: Wikipedia: Online-Filmdatenbank

Dieses Programm kann Inhalt der OFDB über das OFDB GW abfragen.

Folgende Informationen werden abgefragt:
  • Name des Films
  • Erscheinungsjahr
  • Beschreibung / Plot
