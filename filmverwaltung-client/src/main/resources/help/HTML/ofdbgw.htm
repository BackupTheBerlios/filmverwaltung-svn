Was ist das OFDB GW ?

Das OFDB GateWay ist eine XML-Schnittstelle zur OFDB durch einen Dritt-Anbieter.

Sie wird entwickelt und betrieben von Björn Böttcher, der hier ganze Arbeit geleistet hat. Diese Schnittstelle gefällt mir persönlich sehr gut. Das dürfe wohl auch der Grund sein, warum ich sie verwende. ;)

Beispielaufrufe:
  • http://xml.n4rf.net/ofdbgw/search/Austin
  • http://xml.n4rf.net/ofdbgw/movie/1344
Aus diesen XML-Antworten werden Zusatzinfos für die eingetragenen Filme gelesen.
